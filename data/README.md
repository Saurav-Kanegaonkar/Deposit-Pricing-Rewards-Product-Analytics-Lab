# Data Sources

Synthetic source-style data for deposit pricing, rewards engine enhancements, and cross-functional banking product analytics.

- `entities.csv`: 36 deposit segment records.
- `daily_metrics.csv`: 5,040 deposit segment-day metric rows.
- `source_events.csv`: 760 operational events, QA checks, experiments, and stakeholder asks.
- `recommended_actions.csv`: 220 candidate actions with effort and expected lift.
